Repository containing the custom electrical systems I created/help creating.

Currently it contains documentation about:

- Custom datalogger using the Echook core with custom TX module and handheld-based receiver with 4" LCD screen + code
- Custom ESP32-C3 based system to measure current through the motor without adding a shunt resistor + code
- Custom PWM (V5, previous versions were not built by me) with both hardware documentation and code
- Documentation for all custom cables wired through the car + connector pinouts
