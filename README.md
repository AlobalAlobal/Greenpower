Repository containing the custom electrical systems I created/help creating.

Currently it contains documentation about:

- PWM (V1–V5)
- Custom datalogger using the Echook core with custom TX module and handheld-based receiver with 4" LCD screen
- Custom ESP32-C3 based system to measure current through the motor without adding a shunt resistor
- Documentation for all custom cables wired through the car + connector pinouts
